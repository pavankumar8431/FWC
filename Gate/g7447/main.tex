\textbf{Linked Answer Questions}

\textbf{Statement for Linked Answer Questions 52 and 53>}

The following Karnaugh map represents a function $F$.

\vspace{0.4cm}

\begin{center}
\begin{tabular}{c|c c c c}
 & \multicolumn{4}{c}{$YZ$} \\
 & 00 & 01 & 11 & 10 \\
\hline
$X=0$ & 1 & 1 & 1 & 0 \\
$X=1$ & 0 & 0 & 1 & 0 \\
\end{tabular}
\end{center}

\vspace{0.4cm}

\textbf{Q.52} A minimized form of the function $F$ is:

\begin{enumerate}
\item[(A)] $F = \bar{X}Y + YZ$
\item[(B)] $F =\bar{ X}\bar{Y} + YZ$
\item[(C)] $F = \bar{X}\bar{Y} + \bar{Y}Z$
\item[(D)] $F = X\bar{Y} + \bar{Y}Z$
\end{enumerate}

\vspace{0.3cm}
\textbf{Correct Answer: (B)}
%------------------------------------------------
\bluesection{Question Analysis}

The Boolean expression:

\[
F = X'Y' + YZ
\]

The output becomes HIGH when:
\begin{itemize}
    \item Both X and Y are 0
    \item OR both Y and Z are 1
\end{itemize}

\bluesection{Hardware Implementation}

Arduino generates all 8 input combinations (000 to 111).  
The Boolean function is evaluated in Arduino.  
The output is sent as BCD input to 7447.  
7447 drives the common anode 7-segment display.

%------------------------------------------------
\bluesection{Required Components}

\begin{itemize}
    \item Arduino UNO
    \item IC 7447
    \item Common Anode 7-segment display
    \item 220$\Omega$ resistor(s)
    \item Breadboard
    \item Connecting wires
\end{itemize}

%------------------------------------------------
\bluesection{Logic Description}

For each input combination:

\[
F = X'Y' + YZ
\]

If $F = 1$, BCD 0001 is sent to 7447.  

If $F = 0$, BCD 0000 is sent to 7447.  

Thus, the 7-segment display shows either 0 or 1.

%------------------------------------------------
\bluesection{Code Uploading Steps}

\begin{enumerate}
    \item Create a Platform IO project.
    \item Write the code in \texttt{main.cpp} in \texttt{src}.
    \item Run the PIO project with command \texttt{"pio run"}. It will compile the code and create a \texttt{.hex} file.
    \item Copy the \texttt{.hex} file to ArduinoDroid folder.
    \item Connect the Arduino UNO to mobile with OTG cable.
    \item Upload the \texttt{.hex} file using \texttt{"upload precompiled"} option.
    \item Observe the output and verify the expression.
\end{enumerate}
